Versions and Changes

  • Version .92 - 5/19/2015
    • Created the ability for paradigm input pages to offer input for different classes of paradigms
    • Added past tense suffixes to the input and output paradigms
    • Changed the map source to MapQuest instead of OpenStreetMaps since the MapQuest servers are more reliable and maps are more readable
    • Added information to the 'About' page
  • Version .91 - 1/23/2015
    • Added information about which bibliographic sources are used for each dialect on the dialect info page
    • Added 'collaborators' so that private data can be shared with other users. Collaboration relationships are unidirectional. Users may add collaborators through the admin interface.
    • Added this versions and changelog page.
  • Version .90 - 6/25/2015
    • First publicly accessible version of DAD.

This page last updated 6/26/2015
